# Contributing

SoulPrompts is a SvelteKit storefront/CMS project. Contributions should stay small and easy to review.

## Local setup

```bash
npm install
npm run verify
```

The app can run with the in-memory CMS fallback for local exploration.

## Guidelines

- Do not commit secrets or private dashboard credentials.
- Keep demo content fake or clearly licensed.
- Include screenshots for UI changes when possible.
- Run the most relevant checks before opening a PR.
